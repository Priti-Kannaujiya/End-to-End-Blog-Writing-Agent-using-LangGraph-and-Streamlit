# End-to-End-Blog-Writing-Agent-using-LangGraph-and-Streamlit
In this end to end project, I have built an AI-powered Blog Writing Agent using LangGraph, Streamlit, and OpenAI. The agent can take a topic from the user, plan the blog structure, generate well-organized content, include research-based evidence when needed, create image suggestions, and provide a final markdown preview with downloadable files.

## Description

- This project is an AI-powered Blog Writing Agent built using `LangGraph`, `Streamlit`, and `OpenAI`. It takes a topic from the user, creates a blog plan, generates content, adds evidence when needed, generates images for the blog, and provides the final blog in markdown format.

Steps I followed:

1. Built the frontend using `Streamlit`.
2. Created a blog workflow using `LangGraph`.
3. Used `ChatOpenAI` to generate blog plans and content.
4. Added research/evidence support for better accuracy.
5. Added image generation support for blog visuals.
6. Merged all sections into a final markdown blog.
7. Added options to preview and download the blog.
